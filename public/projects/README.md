# Project Images

Place your project screenshots and images in this directory.

Recommended specifications:
- Format: JPG, PNG, or WebP
- Dimensions: 1200x630px (or similar 16:9 aspect ratio)
- Size: Keep under 500KB for optimal loading

## Naming Convention

Use descriptive names for your images:
- ecommerce.jpg
- taskapp.png
- portfolio.jpg
- etc.

Then update the `image` property in your project data (in `src/app/projects/page.tsx`) to reference these images:

```typescript
image: '/projects/your-image.jpg'
```
